# maventestproject

This is baseline maven cucumber framework

[![Build Status](https://travis-ci.org/accesspritam/maventestproject.svg?branch=master)](https://travis-ci.org/accesspritam/maventestproject)

This test repository is integrated with 3 continous integration CI tools namely
1. Travis CI Org (Going to be deprecated) (https://travis-ci.org/github/accesspritam/maventestproject)
2. Travis CI Com (New) (https://travis-ci.com/github/accesspritam/maventestproject) 
3. Circle CI (https://app.circleci.com/pipelines/github/accesspritam/maventestproject?branch=master)
4. Cloudbees Jenkins CI (setup on local)
